# Palindrome Checker Management System

## Overview
A console-based Java application that validates whether a given string is a palindrome under different conditions.

## Use Case 1 — Application Entry & Welcome Message
Displays the application name and version on startup.

## How to Run

### Compile
```bash
javac UseCase1PalindromeCheckerApp.java
```

### Run
```bash
java UseCase1PalindromeCheckerApp
```

### Expected Output
```
Welcome to the Palindrome Checker Management System
Version : 1.0
System initialized successfully.
```

## Key Concepts
- **Class** – Container for application logic
- **Main Method** – Entry point (`public static void main(String[] args)`)
- **Static Keyword** – Allows JVM to invoke `main()` without an object
- **Console Output** – `System.out.println()`
- **Application Flow Control** – Startup behavior before palindrome processing
